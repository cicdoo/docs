## Introduction

**CICDoo** is a managed platform for deploying, running, and maintaining Odoo instances across any number of servers. It provides a unified control plane for the full lifecycle of an Odoo deployment:

- Connect your own Linux servers on any cloud provider over SSH.
- Import repositories from GitHub or GitLab and create fully configured Odoo environments in minutes.
- Manage multiple environments per project (Development, Staging, Production), on the same server or each on its own server.
- Stop, restart, merge, monitor, back up, restore, and troubleshoot instances from a single browser-based console.
- Access a full browser-based VS Code IDE inside each instance.
- Collaborate with a team using fine-grained, role-based permissions at workspace level.
- Configure S3-compatible storage for backups across any cloud provider.

This manual documents every feature available through the CICDoo web interface.
