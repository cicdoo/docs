The **Custom Development** page lets you request dedicated Odoo developer resources on a day-rate basis.

### Selecting a Role

Choose from three developer profiles:

| Role                      | Expertise                                                              |
| ------------------------- | ---------------------------------------------------------------------- |
| **Functional Consultant** | Business process analysis, Odoo configuration, user training           |
| **Technical Consultant**  | Module development, integrations, API work, technical customization    |
| **Project Manager**       | Planning, coordination, stakeholder communication, delivery management |

Each role displays its daily rate.

### Configuring Your Request

| Field            | Description                                                         |
| ---------------- | ------------------------------------------------------------------- |
| Duration         | Number of working days (1–90). Use the +/− buttons or type a value. |
| Start Date       | Earliest available start date (minimum: tomorrow).                  |
| Project Title    | Short title for the engagement.                                     |
| Description      | Detailed requirements, scope, and any technical constraints.        |
| Odoo Version     | Target version.                                                     |
| Edition          | CE or EE.                                                           |
| Email            | Pre-filled from your account — this is where confirmation is sent.  |
| Phone / WhatsApp | Contact number for daily coordination.                              |

### Cost Estimate

A sticky sidebar on the right shows a live cost summary that updates in real time:

- Selected role
- Number of days
- Rate per day
- **Total estimate**

### Submitting a Request

Click **Submit Request**. CICDoo will:

1. Review your requirements within one business day.
2. Match you with a vetted developer from the network.
3. Send a confirmation email with kick-off details.

**Included with every engagement:**

- Background-checked developers
- Satisfaction guarantee
- Daily standups / written progress updates
- NDA and IP transfer agreement
